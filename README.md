# 🦞 OpenClaw Mission Control

一個簡潔的任務追蹤看板，用於管理 OpenClaw 專案的任務進度。

## ✨ 功能特色

- **四欄看板**：待辦事項、進行中、已完成、擱置
- **拖曳操作**：拖曳任務卡片到不同欄位變更狀態
- **優先級標記**：高/中/低三種優先級
- **標籤系統**：為任務添加標籤分類
- **本地儲存**：資料儲存在瀏覽器 localStorage
- **匯入/匯出**：支援 JSON 格式匯入匯出
- **響應式設計**：支援桌面和行動裝置

## 🚀 使用方式

### 線上使用

訪問：https://bylmdada.github.io/openclaw/

### 本地使用

直接用瀏覽器打開 `index.html` 即可使用。

## ⌨️ 快捷鍵

| 按鍵 | 功能 |
|------|------|
| `N` | 新增任務 |
| `Esc` | 關閉彈窗 |
